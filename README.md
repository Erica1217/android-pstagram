# Pstagram
편의점 음식 리뷰 앱
> 편의점 + 인스타그램 = pstagram

## ❗ 중요
**현재 서버주소가 변경되어 정상적으로 실행되지는 않습니다 😱**

## 주요기능
- 로그인/회원가입
- 리뷰 업로드
- 전체리뷰 확인
- 내 리뷰 확인
- 새로고침
